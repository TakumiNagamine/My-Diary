# My Diary

## 概要
このアプリは、自分の気分や日々の出来事を記録し、振り返るために作成しました。  
日々の中で感じたことや考えたことを整理し、メンタルの安定や新たな発想につなげることを目的としています。

## 制作背景
私は普段、思いついたことを忘れてしまったり、友人と話した内容をうまく思い出せないことがあります。  
そこで、自分の考えや感情を言葉として残すことで、後から振り返りやすくしたいと考えました。

また、紙に書く場合は私情にはなりますが、周囲の目が気になったり、継続しづらいと感じることもあります。  
このアプリでは、より気軽に記録できる環境を提供することを目指しました。

## 特徴
- 日々の出来事や感情を記録できる
- 自分の思考を整理できる
- 気軽に振り返りができる

## 使用技術
- HTML / CSS / JavaScript
